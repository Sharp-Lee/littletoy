# Windows程序打包说明

## 方法一：使用PyInstaller（推荐）

### 1. 安装PyInstaller

在Windows系统上，打开命令提示符（CMD）或PowerShell，运行：

```bash
pip install pyinstaller
```

### 2. 打包命令

在脚本所在目录运行以下命令：

```bash
pyinstaller --onefile --name=转粮数据汇总工具 --icon=NONE 汇总脚本.py
```

或者使用更详细的配置：

```bash
pyinstaller --onefile ^
    --name=转粮数据汇总工具 ^
    --windowed ^
    --add-data="requirements.txt;." ^
    汇总脚本.py
```

### 3. 生成的文件位置

打包完成后，可执行文件位于：
```
dist/转粮数据汇总工具.exe
```

## 方法二：使用cx_Freeze

### 1. 安装cx_Freeze

```bash
pip install cx_Freeze
```

### 2. 创建setup.py文件（已提供）

运行：
```bash
python setup.py build
```

## 注意事项

1. **依赖库**：确保所有依赖库都已安装（pandas, openpyxl, xlrd）
2. **文件路径**：打包后的程序会在当前目录查找转粮数据文件
3. **虚拟环境**：建议在虚拟环境中打包，避免包含不必要的库

## 使用打包后的程序

1. 将生成的.exe文件复制到包含转粮数据文件的目录
2. 双击运行.exe文件
3. 程序会自动查找所有转粮数据文件并汇总
